##### [AhnLab] 보안 로그 분류를 위한 인공지능 모델 개발 (v1)
---------------------
기간 : 17.01 ~ 18.04

보안 로그 수집
- 학습을 진행할 데이터 수집. 통합로그시스템(SIEM)으로부터 수집

보안 로그 분류를 위한 레이블링
- 수집된 데이터에 대한 레이블링 수기 진행. 정오탐 수기 분류 작업으로 약 4만건 진행
- 분류해야할 클레스의 개수만큼의 이진 분류 모델을 생성
- 각각의 클레스를 서로 독립적으로 학습하도록 레이블링

모델링
- CNN과 RNN 구조를 합친 AI 모델 디자인
- CNN 영역을 통해 토큰에 대한 피쳐를 학습하고, RNN 영역을 통해 토큰간의 순서를 학습

실제 환경에 모델 적용
- 실제 운영 서버에 deploy할 수 있도록 Tensorflow C++ API를 이용해서 모델 적용 루틴 개발
- 분류해야할 클레스 개수(4개)의 모델에 대해서 각각 이진 모델을 로딩해서 서로 독립적으로 이진 분류를 실행하는 방식으로 적용

※ 이 프로젝트는 18.08-19.09에 진행한 프로젝트에서 아래와 같은 영역을 개선함
- AI 모델 구조 업그레이드(upgrade to BERT-like structure)
- 4개의 독립된 이진 분류 모델이 아닌 단일 multi-label classification 모델로 학습.(하나의 로그가 두 개 이상의 레이블링을 가질 수 있도록 하기 위함)

##### [AhnLab] 이상 행위 사용자 탐지를 위한 머신러닝 모델 개발
---------------------
기간 : 16.10 ~ 17.01

이상 행위를 탐지하기 위한 데이터 레이블링
- 웹 서버의 자원을 급격히 증가시키는 특정 IP의 이상 행위를 포착하여 해당 행위를 레이블링
- 이상 행위에 해당하는 IP는 한 개였음
- 한 개의 IP를 여러 개의 데이터로 만들기 위해 시간 단위로 쪼개어 T에서의 행위와 T+1에서의 행위를 다른 데이터로 구분
- 이상치 데이터를 구하기 어려운 문제를 해결

ML 모델링을 위한 피처 추출
- 이상 행위를 판단하기에 유리한 피처를 도메인 지식을 통해 추출
- 단위 시간당 접근 횟수의 산포도 및 GET/POST 메소드의 비율 등등을 피처로 활용

ML 모델링 개발
- 일반적인 머신러닝 기법을 활용하여 이상행위를 구분하는 이진 분류 모델을 학습
